🏥 Medicare Pro – Your Personal Health Companion
📌 Overview
Medicare Pro is a web-based healthcare management system that helps users monitor their health, track medications, and get AI-based symptom insights.
It provides a simple and user-friendly interface for managing daily health activities efficiently.
🚀 Features
🔐 User Authentication (Login system)
📊 Health Dashboard (Heart rate, Oxygen, Temperature, Glucose)
💡 Daily Health Advice Generator
📈 Heart Rate Graph Visualization
🤖 AI Symptom Checker
💊 Medication Tracker (Mark as taken / pending)
